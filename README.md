  ## Installation Instructions 📣
---

  1. Clone Repo
  2. Run ``` npm i```  in repo directory  
  3. Run  ``` npm start ``` in terminal. 
  
  ## Usage 🖱️
---
A hypothetical use case would be:
```

As a business owner
I want to be able to view and manage the departments, roles, and employees in my company
```
